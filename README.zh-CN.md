## 👋 介绍

这里是 Web Infra 团队的 blogs，我们创建了 Rspack、Rsbuild、Rspress、Rsdoctor、Oxc、Modern.js、Garfish 等项目。

我们创建这个频道的目的是：

1. 透明地开发各个项目，从而构建可信赖的基础设施。我们会优先公开讨论事宜，也会将一些 RFC 发布到这里。
2. 我们还将分享一些工作中遇到的有趣事物，当你遇到类似问题时，这可能会有所帮助。
